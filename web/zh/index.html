免费、开源的 AI 网页自动化工具。

WebBrain 可在 Chrome 与 Firefox 中运行。你可以读取网页、做信息研究，并使用本地 LLM（llama.cpp）或云服务（OpenAI、Claude、OpenRouter）自动执行任务。

安装到 Chrome 安装到 Firefox 查看 GitHub
